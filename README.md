Welcome to SheharBuddy
SheharBuddy is a highly interactive, hyperlocal relocation web application built to solve the challenges of moving to a new city. Designed with a mobile-first, "Swiggy-style" user experience, the platform relies on a community-driven database to surface essential local knowledge—ranging from PG accommodations and transit routes to hospitals and local food gems.

Core Features:

The Filter Matrix & Live Search: A buttery-smooth UI that updates place feeds and category filters instantly on every keystroke without reloading the page.

Relocation Budget Estimator: An interactive toolkit allowing users to calculate their estimated monthly living costs using dynamic sliders and data visualizations.

Gamification Engine: A built-in "Buddy Points" system that rewards users for completing newcomer checklists, adding verified places, and writing local tips, allowing them to climb the leaderboard from "Explorer" to "City Legend."

Custom Dashboard: A highly polished user profile to track contributions, saved places, and community standing.
